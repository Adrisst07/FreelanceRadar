FreelanceRadar
Agregador de vagas freelancer com dashboard responsivo.
 Nota: Dados são simulados para demonstração.
Como Usar
bashgit clone https://github.com/seu-usuario/FreelanceRadar.git
cd FreelanceRadar

python -m venv venv
venv\Scripts\activate

pip install -r requirements.txt

cd backend
python app.py
Acesse: http://localhost:5000
Características

Dashboard responsivo (mobile, tablet, desktop)
Filtros em tempo real por título e plataforma
Dados simulados de 3 plataformas (Upwork, Workana, Fiverr)
API REST para consulta de vagas
Coleta automática a cada 1 hora (APScheduler)
Persistência com SQLite
Bootstrap 5 + JavaScript vanilla

O que Demonstra
 Backend com Flask + API REST
 Frontend responsivo e moderno
 Automação com APScheduler
 Banco de dados SQLite
 Integração frontend-backend
 Boas práticas de código
Tecnologias
Python, Flask, SQLite, HTML, CSS, JavaScript
